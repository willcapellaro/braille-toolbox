# Braille Toolbox SPA

## Overview

The Braille Toolbox is an educational web application designed to serve as a comprehensive reference for braille learning and practical use. This single-page application (SPA) allows users to interactively encode and decode braille, learn braille patterns, and understand Unified English Braille (UEB) standards.

## Features

- **Quick Reference Index**: Easily access braille content and links to various educational resources.
- **Central Search Module**: A powerful search feature that includes a toggling braille cell for enhanced learning.
- **Mobile Navigation**: A responsive hamburger menu for seamless navigation on mobile devices.
- **Bookmarking Support**: Good URL query support for easy bookmarking of specific content.

## Project Structure

The project is organized into the following main directories:

- **public/**: Contains static assets such as the main HTML file, favicon, and braille SVG assets.
- **src/**: The source code for the application, including components, hooks, services, data, styles, and utilities.

## Installation

To get started with the Braille Toolbox SPA, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   cd braille-toolbox-spa
   ```

2. Install the dependencies:
   ```
   npm install
   ```

3. Start the development server:
   ```
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:3000` to view the application.

## Technologies Used

- **React**: For building the user interface.
- **TypeScript**: For type safety and better development experience.
- **Tailwind CSS**: For utility-first styling and responsive design.
- **Vite**: As the build tool for fast development and optimized production builds.
