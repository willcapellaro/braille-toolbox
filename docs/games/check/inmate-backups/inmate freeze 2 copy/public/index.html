How to Play:

Use keys 1, 2, 3, 4, 5, 6 to turn spotlights on/off. Multiple spotlights can be active at once!

Inmates freeze when caught in spotlights! Keep them frozen for 1 second to catch them.

Watch the red timer ring around frozen inmates - when it completes, they're caught!

Keyboard Controls:
1 = Left spotlight, 2-5 = Middle spotlights, 6 = Right spotlight
