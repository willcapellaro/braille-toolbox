Braille Toolbox Font

Beta Product

Feedback

Use the Braille Toolbox Font on your device or website to display contracted braille. It works using ligatures, a font feature that replaces character combinations with new ones. As you type, contractions will automatically appear, and can be edited. Try it for yourself on the Write in Braille page.
